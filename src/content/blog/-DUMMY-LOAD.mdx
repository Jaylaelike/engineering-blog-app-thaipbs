## การตรวจซ่อมชุดระบายความร้อน DUMMY LOAD ยี่ห้อ Blower Assembly Model : BA-300

> โดย : เจ้าหน้าที่ส่วนงานวิศวกรรมชุมพร

![](https://raw.githubusercontent.com/Jaylaelike/blog-template/main/src/assets/images/screenshot-2567-04-24-at-13.31.15.png)

รูปที่ 1 ชุดระบายความร้อน Dummy Load  ยี่ห้อ Blower Assembly Model : BA-300

เนื่องด้วยเจ้าหน้าที่ของส่วนงานวิศวกรรมชุมพร  เข้าทำการซ่อมบำรุงเชิงป้องกัน ([Preventive maintenance)](https://dictionary.sanook.com/search/dict-computer/preventive-maintenance) ที่สถานีส่งสัญญาณฯ ประจวบคีรีขันธ์ได้ทำการทดสอบการทำงานของเครื่องส่งสัญญาณ ขณะทำการทดสอบพบว่า Dummy Load มีอุณหภูมิสูง จึงเข้าทำการตรวจสอบชุดระบายความร้อน Dummy Load ยี่ห้อ Blower Assembly Model : BA-300 ปรากฏว่า มีพัดลมของชุดระบายความร้อนชำรุด 1 ตัวและทำการถอดพัดลมตัวดังกล่าวเพื่อดำเนินการจัดหาอะไหล่ทดแทนและจัดทำคู่มือการตรวจซ่อม เพื่อใช้เป็นข้อมูลและใช้เป็นแนวทางการซ่อมในครั้งต่อไป อีกทั้งยังเป็นการช่วยองค์กรลดค่าใช้จ่ายในการจัดซื้ออุปกรณ์ใหม่ทั้งชุด

1. พัดลมขนาด 7 นิ้ว จำนวน 1 ตัว
2. [ไขควง](https://www.lazada.co.th/tag/%E0%B9%80%E0%B8%81%E0%B8%88%E0%B8%A7%E0%B8%B1%E0%B8%94%E0%B9%81%E0%B8%A3%E0%B8%87%E0%B8%94%E0%B8%B1%E0%B8%99%E0%B8%99%E0%B9%89%E0%B8%B3%E0%B8%A2%E0%B8%B2%E0%B9%81%E0%B8%AD%E0%B8%A3%E0%B9%8C/)และประแจ
3. สว่านพร้อมดอกสว่าน
4. ชุดหัวแร้ง
5. ท่อหด
6. น๊อตสำหรับยึดพัดลม

### ขั้นตอนการซ่อม

ทำการถอดอุปกรณ์ชุดระบายความร้อน Blower Assembly Model : BA-300 แยกออกจากอุปกรณ์ Dummy Load \
![](https://raw.githubusercontent.com/Jaylaelike/blog-template/main/src/assets/images/screenshot-2567-04-24-at-13.40.54.png)

รูปที่ 3 ชุดระบายความร้อน Blower Assembly ติดตั้งร่วมกับอุปกรณ์ Dummy Load

1. จากนั้นทำการถอดตะแกรงป้องกันฝุ่นและพัดลมตัวที่ชำรุดออกพร้อมทำความสะอาด![](https://raw.githubusercontent.com/Jaylaelike/blog-template/main/src/assets/images/screenshot-2567-04-24-at-13.41.10.png)
2. นำพัดลมตัวใหม่มาทำการทดลองติดตั้งเพื่อทำการมาร์คตำแหน่งน๊อตสำหรับเจาะยึด (เนื่องจากเป็นอะไหล่ทดแทนอะไหล่เดิมที่ไม่มีขายในท้องตลาด)\
   ![](https://raw.githubusercontent.com/Jaylaelike/blog-template/main/src/assets/images/screenshot-2567-04-24-at-13.48.36.png)
3. ใช้สว่านเจาะให้มีขนาดรูเท่ากับน๊อตที่จะใช้ในการยึดพัดลมทั้ง 4 มุมเพื่อความแข็งแรงในการติดตั้ง
